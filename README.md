# KeraloxBlowdownSim2026
Python based blowdown simulation of the Keralox vehicle being developed in 2026.

Further info can be found in the Wiki.

# Setup Prior to Use
REQUIRED LIBRARIES:
`pandas`
`matplotlib`
`seaborn`

I use a conda environment and installed them using `pip`.
